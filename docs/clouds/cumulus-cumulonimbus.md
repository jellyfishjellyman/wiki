## 总览

积云和积雨云是低云族中最典型的对流云。积云从低层暖湿空气的局地抬升开始，常表现为云底较平、云顶向上凸起、轮廓清楚的云块；当上升运动增强，云体迅速增厚增高，并发展到冰晶化高度后，积云可转化为积雨云。成熟积雨云常贯穿低、中、高多个高度层，云顶形成鬃状或砧状冰晶结构，云底则可出现降水带、雨幡、碎雨云、悬球状云底、弧状云或管状云等附属特征。

《中国云图》的云类编码把淡积云、碎积云归入 `CL1`，浓积云归入 `CL2`，秃积雨云归入 `CL3`，鬃积雨云和多数成熟积雨云归入 `CL9`。这套编码服务于地面观测云码，和国际十云属体系不是同一层级；本页同时保留中文云类、云属缩写和《中国云图》云码。

## 形成机制

积云的基本触发机制是低层空气抬升、冷却并达到凝结高度。常见触发包括地面日照加热、海陆风或陆风环流、山地抬升、低压或冷锋前沿的动力抬升。云底高度大致对应凝结高度，因此一片积云群的云底常接近同一水平面；云顶高度反映上升气流强弱、环境层结和高空风。

从《中国云图》图版看，积云发展受地形和局地环流影响明显：八达岭图 6 中暖湿空气受山地抬升形成碎积云；桂林图 7 中山顶上空初生淡积云底部受山地气流影响略向上倾斜；海南崖县图 14 中淡积云受陆风影响由岸上向海面移动；西沙群岛图 25 中海面中午前后常有浓积云排列成行；多庆错图 23 中雪山附近常生成浓积云，云底部很平整且距地面约 50 米。

浓积云继续发展时，如果上升气流足够强，可迫使云顶上方较潮湿气层迅速冷却凝结，形成覆盖在云顶上的薄幕状附属云。《中国云图》图 24 称其为“帽状云”；《云彩收集者手册》对应的国际术语为幞状云 `pileus`，强调它常短暂出现在迅速向上发展的积云顶部。两者属于同一类或高度相近的概念，术语对应关系仍需后续校订。

当云顶达到足够低温并开始冰晶化，清晰的水滴云泡会逐渐转为模糊、纤维状或毛丝状结构；高空风会把云顶水平展开，形成云砧。云内强上升、下沉气流并存时，云底会变黑、起伏、混乱，并可出现降水带、雨幡、泄雹带或悬球状云底。

## 淡积云

| 项目 | 内容 |
| --- | --- |
| 所属分类 | 低云族，积云属；《中国云图》云类代码 `CL1` |
| 英文 / 拉丁名 | Cumulus humilis |
| 定义 | 垂直发展较弱、个体较小的积云，云顶呈圆弧形凸起，云底多较平。 |
| 识别特征 | 白色云块，边缘可稍破碎；云底有阴影；水平宽度通常大于垂直厚度；个体分明，常散布于蓝天。 |
| 形成机制 | 弱到中等的热力对流、山地抬升、海陆风或陆风环流触发。 |
| 天气意义 | 多为晴好天气中的对流云；若云体迅速增厚增高，需继续观察是否向浓积云发展。 |
| 易混淆对象 | 碎积云、积云性层积云、远处浓积云。判断时看云块是否完整、云底是否平整、垂直发展是否明显。 |
| 观测要点 | 记录出现时间、日变化、移动方向、云底是否一致、云体是否连接或仍个体分明。 |

## 碎积云

| 项目 | 内容 |
| --- | --- |
| 所属分类 | 低云族，积云属；《中国云图》云类代码 `CL1` |
| 英文 / 拉丁名 | Fractocumulus，常简写 `Fc` |
| 定义 | 破碎、不规则、轮廓不完整的小积云云块。 |
| 识别特征 | 边缘破碎，大小不匀，形状多变，变化速度较快；可零散分布，也可逐渐长大成为淡积云。 |
| 形成机制 | 弱对流初生阶段、山地抬升、局地湍流或原有积云破碎。 |
| 天气意义 | 单独出现常天气较好；若在积雨云或降水云底附近出现，则可能与降水蒸发、局地凝结有关，需要区别碎雨云。 |
| 易混淆对象 | 碎层云、碎雨云、远处淡积云。碎积云多有积状凸起或处在积云发展链条中；碎雨云常在降水云底。 |
| 观测要点 | 连续观察是否长大、合并、云底是否趋于平整，以及是否受地形或局地风影响。 |

## 浓积云

| 项目 | 内容 |
| --- | --- |
| 所属分类 | 低云族，积云属；《中国云图》云类代码 `CL2` |
| 英文 / 拉丁名 | Cumulus congestus |
| 定义 | 垂直发展旺盛的积云，云体高大，云顶明显向上隆起，常呈花椰菜状。 |
| 识别特征 | 垂直高度大于或接近水平宽度；云顶对流泡体清楚；云底较平且呈暗灰色；强发展时可有雨幡或帽状云。 |
| 形成机制 | 强热力对流、地形抬升、冷锋前沿或低压系统触发，云中上升气流强。 |
| 天气意义 | 是积雨云的前阶段之一。若迅速增高、云底变黑、有雨幡或帽状云，应警惕雷阵雨、冰雹或短时大风。 |
| 易混淆对象 | 淡积云、秃积雨云。关键看垂直发展强度和云顶是否已冰晶化：浓积云顶部仍多为清晰水滴云泡，秃积雨云云顶开始变毛、变模糊。 |
| 观测要点 | 记录云顶增高速度、云泡是否倾斜、云底是否出现雨幡、是否与其他单体合并、上方是否出现幞状/帽状云。 |

## 秃积雨云

| 项目 | 内容 |
| --- | --- |
| 所属分类 | 低云族，积雨云属；《中国云图》云类代码 `CL3` |
| 英文 / 拉丁名 | Cumulonimbus calvus |
| 定义 | 从浓积云发展而来、云顶已开始冰晶化但毛丝状结构尚不充分的积雨云。 |
| 识别特征 | 云体庞大，云顶轮廓由清晰云泡转向模糊平滑或伸展；云底黑灰，常将出现降雨。 |

| 易混淆对象 | 浓积云和鬃积雨云。浓积云云顶仍有清楚圆拱，鬃积雨云云顶则有明显毛丝、鬃状或砧状冰晶结构。 |

## 鬃积雨云、云砧和成熟积雨云

| 项目 | 内容 |
| --- | --- |
| 所属分类 | 低云族，积雨云属；《中国云图》云类代码多为 `CL9` |
| 英文 / 拉丁名 | Cumulonimbus capillatus；云砧对应附属特征 `incus` |
| 定义 | 云顶完全或明显冰晶化，呈毛丝状、鬃状或水平展开成砧状的成熟积雨云。 |
| 识别特征 | 云体庞大，云顶毛丝般结构明显；云砧可向高空风下游伸展；云底常黑暗、起伏不平，伴降水。 |
| 天气意义 | 常伴雷雨、大风、强阵雨、冰雹；云底旋转或管状云发展时需警惕龙卷。 |
| 易混淆对象 | 密卷云、伪卷云、浓积云顶部。判断时要结合下方对流主体、云砧与云底降水结构。 |

## 悬球状云底

悬球状云底是从云底下垂的一组圆球形或囊袋状云泡。《中国云图》把它作为积雨云云底现象记录，强调下沉气流和上升气流共同作用；《云彩收集者手册》对应 `mamma`，又称乳状云，说明它可见于多种云，但最常被积雨云云砧底部展示出来。两书共同指向“云底下垂囊袋状结构”，但出现云类范围的表述不同，需在术语表中保留差异。

## 弧状云、管状云和龙卷风险

《云彩收集者手册》把弧状云 `arcus` 描述为风暴云前缘的低而长的附属特征，管状云 `tuba` 则是风暴云底部向下伸出的漏斗状附属特征；二者常和积雨云、强阵风、风暴系统有关。《中国云图》低云图版第一批图 2-52 未单列弧状云或管状云，但天气现象图版中有积雨云底部龙卷案例：海南海口图 218、甘肃岷县图 219、佛罗里达图 278。后续应把这些并入“云与天气现象”和“特殊云形”页。

## 观测工作流

1. 先判定是否为积状对流云：看云底是否较平、云顶是否凸起、云块是否个体分明。
2. 判断发展强度：水平宽度大于垂直厚度多为淡积云；垂直高度显著增加、花椰菜云顶清楚则为浓积云；云顶冰晶化、变毛或展开成砧状则进入积雨云阶段。
3. 连续记录时间：积云向积雨云转化需要看增高速度和云顶形态变化，单张照片容易误判。
4. 记录天气背景：低压、冷锋、山地抬升、海陆风、陆风和午后加热都可能解释同一云类的不同形态。
5. 关注强天气信号：云底变黑、降水带下垂、雨幡及地、云砧迅速扩展、悬球状云底、弧状云、管状云、雷声或闪电。

## 易混淆辨析

| 对象 | 区分要点 |
| --- | --- |
| 淡积云 vs 碎积云 | 淡积云云块较完整、云底较平；碎积云边缘破碎、轮廓不完整、变化更快。 |
| 浓积云 vs 秃积雨云 | 浓积云云顶水滴云泡清楚；秃积雨云云顶开始冰晶化，边缘变模糊并向外伸展。 |
| 秃积雨云 vs 鬃积雨云 | 秃积雨云尚未形成明显毛丝状鬃结构；鬃积雨云云顶毛丝、鬃状或砧状明显。 |
| 雨幡 vs 降水带 | 雨幡可不及地，常呈丝状或帘状；降水带通常更厚重并可与地面或海面相接。 |
| 悬球状云底 vs 普通低云碎片 | 悬球状云底呈一组圆球形囊袋并从云底下垂，常在积雨云底或云砧底部；普通碎云形态不规则且不成囊袋状排列。 |
