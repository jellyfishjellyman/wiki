# 积云与积雨云

## 定义

积云和积雨云是典型的对流云。积云从较弱的低层对流开始，云底较平、顶部凸起；当对流强烈发展，云体庞大并出现冰晶化云顶时，可发展为积雨云。

## 发展序列

| 阶段 | 典型云类 | 识别特征 |
| --- | --- | --- |
| 初生或破碎 | 碎积云 | 小而破碎，形状多变 |
| 弱对流 | 淡积云 | 个体较小，水平宽度大于垂直厚度 |
| 强对流 | 浓积云 | 垂直发展旺盛，顶部花椰菜状 |
| 过渡 | 秃积雨云 | 顶部开始冰晶化，轮廓变模糊 |
| 成熟 | 鬃积雨云 | 云顶毛丝状、鬃状或砧状，常伴强天气 |

## 形成机制

地面加热、低层暖湿空气抬升、地形抬升或锋面触发都可增强对流。浓积云继续向上发展并达到冰晶化高度后，云顶结构会从清晰圆拱转为毛丝状或砧状。

## 天气意义

淡积云通常指示对流较弱。浓积云若快速增高、云底变暗并出现雨幡，需警惕雷阵雨。积雨云可产生阵雨、雷电、大风、冰雹，偶有龙卷。

## 观测要点

- 记录云体是否迅速增高。
- 观察云顶是否开始发毛、冰晶化或展开成砧。
- 观察云底是否黑暗混乱，是否有雨幡或降水带。
- 连续记录移动方向和发展时间，比单张照片更有判断价值。
