# 云的分类体系

## 分类原则

云的分类通常先按云底高度分为低云、中云、高云三族，再按宏观形态、物理结构和成因划分为云属、云类。《中国云图》采用“三族、十属、二十九类云状”的整理方式。

## 三族十属

| 云族 | 主要云属 | 一般云底高度 |
| --- | --- | --- |
| 低云 | 积云、积雨云、层积云、层云、雨层云 | 多低于 2500 米 |
| 中云 | 高层云、高积云 | 一般 2500-5000 米 |
| 高云 | 卷云、卷层云、卷积云 | 一般 5000 米以上 |

高度界限会随季节、纬度、地形和天气条件变化，高原地区的高云云底可相对较低。

| 云族 | 云属 | 云类 | 简写 |
| --- | --- | --- | --- |
| 低云 | 积云 | 淡积云、碎积云、浓积云 | Cu hum、Fc、Cu cong |
| 低云 | 积雨云 | 秃积雨云、鬃积雨云 | Cb calv、Cb cap |
| 低云 | 层积云 | 透光层积云、蔽光层积云、积云性层积云、堡状层积云、荚状层积云 | Sc tra、Sc op、Sc cug、Sc cast、Sc lent |
| 低云 | 层云 | 层云、碎层云 | St、Fs |
| 低云 | 雨层云 | 雨层云、碎雨云 | Ns、Fn |
| 中云 | 高层云 | 透光高层云、蔽光高层云 | As tra、As op |
| 中云 | 高积云 | 透光高积云、蔽光高积云、荚状高积云、积云性高积云、絮状高积云、堡状高积云 | Ac tra、Ac op、Ac lent、Ac cug、Ac flo、Ac cast |
| 高云 | 卷云 | 毛卷云、密卷云、伪卷云、钩卷云 | Ci fil、Ci dens、Ci not、Ci unc |

## 识别顺序

1. 先判断云底高度和主体云族。
2. 再判断外形是积状、层状、纤维状还是波状。
3. 观察透光性、云底、云顶和边缘结构。
4. 结合演变过程判断云类，例如浓积云是否继续发展为积雨云。
