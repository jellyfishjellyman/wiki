## 范围和分类位置

特殊云形不是一个与低云、中云、高云并列的高度分类。它更像一组横切分类：同一种形态可能附着在不同云属上，也可能由地形、飞行活动或特殊观测视角造成。整理时应同时记录母云、云族、高度、天气背景和图版来源。

| 类型 | 分类位置 | 记录重点 | 本 Wiki 关联页 |
| --- | --- | --- | --- |

## 堡状云

堡状云的关键不在“像城堡”这一比喻，而在云层顶部出现一串向上发展的塔状凸起，说明该高度层内有局地对流或不稳定增强。它可以出现在高积云中，也可出现在层积云中。若堡状云在清晨或上午出现，常被用作对流天气潜势的经验信号，但不能脱离湿度、风切变和天气系统单独判断。

| 项目 | 内容 |
| --- | --- |
| 所属分类 | 云种或形态特征，常见于高积云、层积云。 |
| 识别特征 | 云条或云层基部相对水平，顶部有一排塔状、城垛状突起。 |
| 形成机制 | 中层或低层云层顶部出现不稳定，局地上升气流突破原本较平整的云顶。 |
| 天气意义 | 可提示该层不稳定和对流发展可能性，尤其需与高积云、积雨云发展同看。 |
| 易混淆对象 | 絮状高积云、浓积云、积云性层积云。堡状云通常仍有较连续的基部或排列。 |
| 典型图版 | 《中国云图》图 106-111 为堡状高积云；层积云中的堡状类型后续需继续从原书分类表和图版中校订。 |

## 荚状云

荚状云呈豆荚、透镜或飞碟状，通常中间厚、边缘薄。它既可以作为高积云或层积云的细分类，也可以作为地形波云的重要外观。判断时应记录是否在山脉下风侧、是否多层叠置、是否近乎静止，以及云体边缘是否光滑。

## 悬球状云底 / 乳状云

悬球状云底是云底向下垂挂的一组圆球状或囊袋状结构。《中国云图》在积雨云部分给出图 38-40，强调它出现于积雨云云底，并与强对流云内上升、下沉气流的相互作用有关。《云彩收集者手册》对应 `mamma`，常译作乳状云或悬球状云，指出它尤其常见于积雨云云砧下方，但并不限于积雨云。

| 项目 | 内容 |
| --- | --- |
| 所属分类 | 附属特征，常附着在积雨云云底或云砧底面。 |
| 识别特征 | 多个圆球、囊袋、乳房状下垂体，底部轮廓清楚。 |
| 天气意义 | 常提示强对流云体成熟或消散阶段附近存在复杂下沉和湍流；不能单独等同于龙卷或冰雹。 |
| 易混淆对象 | 碎雨云、低云破片、降水帘。区别在于悬球状云底为囊袋状附着结构，不是破碎云块或降水条带。 |
| 术语待校订 | “悬球状云底”“悬球状云”“乳状云”和 `mamma` 的中文规范译名需要对照 WMO 与三本书统一。 |

## 雨幡和雪幡

雨幡或雪幡是降水粒子从云中落下形成的丝状、帘状、拖尾状结构。若低层空气干燥，降水可能在到达地面前蒸发或升华，因此只形成空中下垂的幡状尾迹。

## 幞状云、帽状云和缟状云

《中国云图》图 24 使用“帽状云”描述浓积云顶部上方的轻纱状薄云：强上升气流使云顶上方的湿层快速冷却凝结，像帽子一样覆在发展中的云泡上。《云彩收集者手册》对应 `pileus`，中文常见译名为幞状云。二者高度相近，但术语是否完全等同需继续校订。

`velum` 常译作缟状云或幕状附属云，表现为更宽广的薄幕状结构，和强对流上升气流附近的饱和层有关。当前在《中国云图》已整理图版中尚未定位可靠对应图号；暂保留为 EPUB 线索，不写入确定图版字段。

## 弧状云、管状云和龙卷相关云形

弧状云 `arcus` 通常位于积雨云或风暴系统前缘，呈低而长的弧形、滚轴状或陆架状结构，常与阵风锋相关。管状云 `tuba` 是从云底向下伸出的漏斗状附属云；若旋转漏斗发展并接触地面或水面，可进入龙卷、水龙卷等天气现象讨论。

《中国云图》低云图版第一批未单列弧状云和管状云，但补充图版 [图 278](china-cloud-atlas/plates/supplementary-cloud-weather-examples.md) 记录佛罗里达积雨云底部在海面形成龙卷，说明“漏斗状云”和强烈旋转风暴的关系。天气现象图版中另有国内龙卷图版，后续应在 [云与天气现象](weather-phenomena.md) 中与本节互链。

## 飞机尾迹和人为云

飞机尾迹是高空飞机发动机排出水汽和微粒后，在低温、高湿环境中形成的线状冰晶云。《中国云图》图 244 记录山东泰山气象站观测到飞机尾迹，图 245 记录北京西部特技飞行后，尾迹经过 10-15 分钟扩展成粗条状。

| 项目 | 内容 |
| --- | --- |
| 识别特征 | 初始为细长白线，常成对或多条平行；在适宜湿度下会扩展、弯曲、变宽。 |
| 形成条件 | 高空低温、水汽达到或接近冰饱和、飞机排放提供水汽和凝结/凝华核。 |
| 天气意义 | 可指示飞行高度附近湿度较高；持续并扩散的尾迹会增加高云状覆盖。 |
| 易混淆对象 | 毛卷云、密卷云、辐辏状卷云。尾迹常有直线几何形和飞行路径关系。 |
| 待校订 | `contrail`、`distrail`、中文“航迹云”“飞机尾迹”“耗散尾迹”的对应关系需对照 EPUB 和 WMO 术语。 |

《云彩收集者手册》补充了两个需要单独保留的点：航迹云能否形成和持续，取决于飞机巡航高度附近是否足够冷、足够湿；如果尾迹持续数小时，冰晶会从周围空气继续吸收水汽并扩散，可能促成卷云、卷积云或卷层云状覆盖。相反，`distrail` 是飞机穿过云层后造成的线状空隙，可能由发动机热量和湍流使云滴蒸发，也可能由飞机排放冰核促使过冷液滴冻结下落形成。上述机制来自 EPUB 章节抽取，具体图例地点和摄影者待校订。
