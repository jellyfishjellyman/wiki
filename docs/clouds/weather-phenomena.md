## 降水粒子和云内微物理

降水粒子页补充了云本身的微观组成：云滴、过冷水滴、冰晶、雪晶、霰、冻滴、雨滴、冰雹和雪。它们解释了为什么积雨云能产生冰雹，为什么卷层云能产生晕，为什么雾凇和飞机积冰常与过冷水滴有关。

## 雾、轻雾和霾

雾是近地面水汽凝结形成的悬浮小水滴或冰晶集合，和层云关系密切：雾抬升可成为层云，层云云底降至地面也可表现为雾。霾则以悬浮尘粒为主，不能简单归为雾。

## 观测记录字段

天气现象比普通云类更依赖过程记录。建议每次记录至少包含：

| 字段 | 说明 |
| --- | --- |
| 现象名称 | 例如虹、霓、辐射雾、线状闪电、雾凇、冰雹、沙尘暴。 |
| 母云或环境 | 积雨云、卷层云、高积云、雨层云、过冷云区、江面、海面、逆温层等。 |
| 地点、时间、方向 | 光象和强对流尤其需要太阳方位、云体移动方向、拍摄方向。 |
| 过程 | 是否正在降雨、是否移过测站、是否增强或减弱、持续时间。 |
| 量级 | 能见度、积雪深度、积冰厚度、冰雹直径、雾凇宽度等。 |
| 色带和形态 | 光象需记色带顺序；闪电需记接地/云中/枝状；雾凇需记晶状、粒状、针状。 |
| 来源和校订状态 | 图号、PDF 页、原书页、摄影者、OCR 是否可靠、EPUB 是否待校订。 |
