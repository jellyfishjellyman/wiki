# 云的观测方法

## 观测目标

云的观测应记录云的种类、云量、高度、方向、移动、演变和伴随天气。对于个人 Wiki，重点是让每张图片能被复核：看得见图、找得到来源、知道哪些判断可靠。

## 记录要素

| 要素 | 建议记录 |
| --- | --- |
| 时间 | 年月日、时分，注明时区或地方时 |
| 地点 | 城市、地名，必要时记录海拔 |
| 方向 | 面向方位，如 N、NE、SW |
| 云类 | 中文名、缩写、必要时标注不确定 |
| 高度 | 可估计云底高度或标注“待校订” |
| 天气 | 降水、雷电、风、能见度、光象等 |
| 演变 | 云体增厚、消散、移动、发展成其他云类 |
| 来源 | 原书、图号、页码、摄影者或文件路径 |

## 识别流程

1. 先看整体高度和天空占比。
2. 再看形态：积状、层状、纤维状、波状或特殊形态。
3. 判断透光性和云底结构。
4. 查图例库进行对照。
5. 对无法确认的名称写“待校订”，不要强行归类。

## OCR 和扫描页校订

OCR 文本只能作为线索。遇到图号、地点、时间、方向、拍摄者、云类代码等关键信息，应优先核对原始页面图像或已人工校订的结构化图版页。

## 图片引用规范

主题页可以引用 `docs/assets/images/clouds/china-cloud-atlas/` 下的 webp 图像，但正文应解释识别要点，而不是只复刻原书页。原始扫描页和 OCR 页保留在《中国云图》档案中。
