# 云的观测方法

!!! note "第七批整理范围"
    本页把《中国云图》前言、文字说明、图版字段和已整理图版页中的做法抽象成 Wiki 观测规范：如何记录一张云图，如何标注云类、云码、地点、时间、方向、天气意义和校订状态。两本 EPUB 的观察型案例后续会继续并入，本页先建立可复核的字段框架。

## 观测目标

云的观测不是只给照片取一个名字，而是把“看到的天空状态”转化为可复核资料。一个合格条目应让后续读者能够回答：

- 图中云是什么，判断依据是什么。
- 它属于哪个云族、云属、云类，是否有 `CL/CM/CH` 云码。
- 它出现在哪里、什么时候、面向什么方向。
- 它是否正在发展、消散、增厚、移入或移出测站。
- 它与降水、雷电、雾、光象、沙尘、地形或飞机观测视角有什么关系。
- 哪些字段来自原书，哪些来自 OCR，哪些仍是“待校订”。

## 云量和覆盖

当前主题页尚未建立完整云量观测规范；在资料整理阶段至少记录“覆盖范围”和“是否布满全天”。《中国云图》图注常用“布满全天”“移至测站上空”“分散在天空”“系统侵入天空”等表述，这些词和云码判断有关，尤其是 `CM5`、`CH5`、`CH6`、`CH7`。

| 表述 | 含义 | 记录方式 |
| --- | --- | --- |
| 分散在天空 | 云量不大，云块彼此分离 | 记录云块方向、是否同一高度 |
| 成行 / 成带 / 成层 | 云体按带状或层状组织 | 记录排列方向和移动方向 |
| 布满全天 | 云层覆盖整个天空 | 记录云属、透光性、云底和伴随天气 |
| 系统侵入天空 | 云系从某方向逐渐扩展进入 | 记录前缘方向、高度角、增厚趋势 |
| 混乱天空 | 多高度、多形态云类混杂 | 记录主要云类组合，避免强行单一归类 |

若后续需要采用八分量或十分量云量，应另建规范并说明与当前图版字段的关系；不要从照片主观估算精确云量后写成确定事实。

## 高度记录

云底高度是云族判断的重要线索，但照片中常无法直接测量。处理原则：

| 情况 | 记录方式 | 例子 |
| --- | --- | --- |
| 原书给出高度 | 原样保留，注明是云底、云顶还是飞行高度 | 图 242：云底约 3000 米，云顶约 6000 米 |
| 飞机上观测 | 记录飞行高度和观测对象 | 图 233：飞机高度 9000 米，观测积雨云顶部 |
| 山地遮蔽 | 记录山体、云底与地形关系 | 图 62：远处山顶被云底遮盖 |
| 只可按云族估计 | 写“低云/中云/高云”，高度待校订 | 层积云，云底高度待校订 |
| 图版无高度字段 | 不补造数字 | 原页未列出 |

高原地区和冬季云底高度可能与平原经验不同；例如《中国云图》说明高云在高原地区可相对较低。主题页应保留地点和地形条件，不用固定高度替代实际观察。

## 方向、移动和时间序列

方向字段在云图中有两层含义：拍摄方向和云体移动方向。两者不能混写。

| 字段 | 说明 | 例子 |
| --- | --- | --- |
| 拍摄方向 | 摄影者面对的方位，常以 N、NE、E 等标注 | 图 159：拍摄方向 E |
| 移动方向 | 云体或系统移动路径，常在说明文字中出现 | 图 161：积雨云由西北向东南移动 |
| 前缘方向 | 云系侵入天空时的前缘位置或高度角 | `CH5`、`CH6` 与卷层云幕前缘高度角相关 |
| 时间序列 | 同一过程不同时间的连续照片 | 图 30、图 218-220、图 224-227 |

连续过程比单张照片更有价值。例如图 30 记录浓积云在约 85 分钟内发展成冰雹云；图 218-220 记录海南海口同一次龙卷过程由粗细和长短变化到消失；图 224-227 记录沙尘暴移至测站上空并导致能见度恶化。

## 识别流程

1. 先确认是否是云，还是雾、霾、沙尘、极光、蜃景等非云现象。
2. 分别判断低云、中云、高云是否存在，不要因为主云醒目而忽略其他云层。
3. 判断云属：积状、层状、纤维状、波状、透镜状、破碎状。
4. 判断细分云类：透光/蔽光、淡/浓、秃/鬃、堡状/絮状/荚状等。
5. 判断云码：按 [分类体系](classification.md) 中的 `CL/CM/CH` 表，而不是直接从云类名机械套码。
6. 检查附属云和天气现象：云砧、雨幡、雪幡、悬球状云底、晕、华、虹、雷电、冰雹、雾凇等。
7. 写出识别依据和易混淆对象。
8. 标记来源和校订状态。

## 易混淆记录

| 易混淆对象 | 区分要点 | 记录建议 |
| --- | --- | --- |
| 淡积云 vs 碎积云 | 淡积云云底较平、顶部圆弧；碎积云更小、更破碎、形状多变 | 可写“淡积云或碎积云，待校订” |
| 浓积云 vs 秃积雨云 | 秃积雨云云顶开始冰晶化、轮廓变模糊 | 记录云顶是否有毛状云丝、是否已有阵性降水 |
| 鬃积雨云 vs 伪卷云 | 鬃积雨云有母体强对流；伪卷云可为云砧残余 | 记录是否可见积雨云主体 |
| 层云 vs 雾 | 雾接地；层云云底不接地但很低 | 记录山体、建筑是否被云底遮蔽和能见度 |
| 层积云 vs 高积云 | 视角大小、云底高度、云块大小和阴影不同 | 若缺高度，保留“层积云/高积云待校订” |
| 卷积云 vs 高积云 | 卷积云云块更小、更高、无明显阴影；高积云块较大 | 可用手指视角规则作为 EPUB 线索，待校订 |
| 晕 vs 华 | 晕多由卷层云冰晶折射/反射形成，角半径较大；华靠近日月，多由小水滴/冰晶衍射 | 记录色带顺序、与太阳/月亮距离 |
| 雾凇 vs 霜 | 雾凇依赖过冷雾滴/云雾；霜多由近地水汽凝华 | 记录是否有雾、风向和迎风面增长 |
| 龙卷 vs 尘卷风 | 龙卷与积雨云云底相连；尘卷风由地面热力旋涡卷尘 | 记录是否有母云、是否接地 |

## 主题页引用规范

主题页不是扫描版复刻，但引用图版时应保留足够溯源字段。

| 引用场景 | 最低字段 | 推荐字段 |
| --- | --- | --- |
| 典型图版表 | 图号、标题、入口、识别要点 | 地点、时间、方向、拍摄者、云码、天气意义 |
| 机制说明 | 图号、现象、机制摘要 | 与其他图版的演变关系 |
| 天气个例 | 图号、天气过程、时间地点 | 能见度、冰雹直径、积冰厚度、持续时间 |
| EPUB 概念 | 书名、概念名、待校订说明 | 章节、图片、摄影者、页码，待系统抽取 |

不要把 OCR 中不可靠的地点、时间或摄影者直接写入确定字段。若只有 OCR 线索，应写“待校订”。

## EPUB 图片来源页校订规则

《一天一朵云》`text/part0016.html` 是图片来源页，并明确说明文内页码对应中文版纸书页码。后续把 EPUB 个例转入主题页时，应按以下规则处理：

| 校订动作 | 可确认内容 | 不能直接确认的内容 |
| --- | --- | --- |
| 搜索摄影者中文名 / 英文名 | 可获得纸书页码线索、摄影者或机构来源线索 | 不能自动确认图片路径、图注边界、授权状态 |
| 对照正文文件 | 可确认案例主题、地点、相邻说明和机制摘要 | 连续图文流中仍可能出现一段文字对应多图或上下图 |
| 写入主题页 | 可写“纸书页码线索”“来源页线索” | 不应写成“已校订图片字段”，除非已核对图片和图注 |

已开始校订的例子包括：黛西·道森糙面云页码 264、伊丽莎白·弗雷豪特彩虹轮页码 265、露西·库克尼悬球云页码 302、吉利恩·埃德金斯碎层云页码 340 等。完整记录见 [EPUB 图文案例索引](epub-case-index.md) 的“图片来源页初步校订表”。

## 个人观察建议

对于未来自有照片，可以按“轻量”和“完整”两级记录：

| 级别 | 字段 |
| --- | --- |
| 轻量记录 | 时间、地点、方向、云类初判、天气、照片路径、待校订项 |
| 完整记录 | 加上云族、云属、云类、云码、云量、云底高度、移动方向、演变过程、识别依据、易混淆对象、来源链接 |

即使是轻量记录，也应保留“不确定”。例如“高积云或卷积云，待校订”比误写成确定的“卷积云”更有价值。
