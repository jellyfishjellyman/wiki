# 云的基础概念

## 定义

云是大气中水汽凝结或凝华后形成的可见聚合体，通常由微小水滴、过冷水滴、冰晶、雪晶或它们的混合物组成。

云不是静态物体，而是大气运动、水汽条件、温度结构、凝结核和冰核共同作用的结果。它的形态、云量、分布和演变，可以反映当时大气的稳定程度、水汽分布和垂直运动状况。

## 形成机制

云形成的基本路径是：含水汽空气被抬升或冷却，达到饱和后，水汽在凝结核上凝结成水滴，或在低温条件下凝华成冰晶。常见触发包括：

- 对流抬升：地面受热后，低层空气上升，常形成积云、浓积云，发展旺盛时可形成积雨云。
- 锋面抬升：暖湿空气沿锋面缓慢滑升，常形成高层云、雨层云等范围较大的层状云。
- 地形抬升：气流越过山地时被迫上升，可形成地形云、旗云、荚状云等。
- 辐射冷却和湍流混合：近地层冷却或混合，可形成层云、碎层云，也可由雾抬升而来。
- 波动和风切变：逆温层附近的波动容易形成波状高积云、层积云或荚状云。

## 识别特征

识别云时应同时观察多个维度，而不是只凭单一形状判断。

| 维度 | 观察要点 |
| --- | --- |
| 高度 | 云底大致属于低云、中云还是高云 |
| 外形 | 是团块状、层幕状、纤维状、波状、堡状还是豆荚状 |
| 结构 | 边缘是否清楚，云底是否平整，顶部是否向上发展 |
| 透光性 | 是否能辨认日月位置，是否可见蓝天或阴影 |
| 演变 | 云量、厚度、移动方向和发展速度是否变化 |
| 伴随现象 | 是否有降水、雨幡、雷电、晕、华、虹等 |

## 天气意义

云的天气意义取决于云类和演变趋势。例如，淡积云常见于晴朗天气；浓积云继续发展可能转为积雨云；雨层云多对应连续性降水；卷层云逐渐增厚并降低，常提示天气系统接近。

!!! warning "避免过度推断"
    单张云图不能单独决定天气预报。应结合时间连续变化、风向、气压、湿度、雷达或卫星资料综合判断。
