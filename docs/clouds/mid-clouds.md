## 定义和范围

中云是云底处于中等高度的一组云。《中国云图》给出的一般云底高度为 2500-5000 米；《云彩收集者手册》译本中常把中云族云底写作约 2000-7000 米。两套高度阈值服务的分类语境不同，本 Wiki 暂以《中国云图》作为主题页主体系，同时在分类页保留差异说明。

中云通常由微小水滴、过冷水滴组成，也可与冰晶、雪晶混合。中云的天气意义比外观更需要结合演变：高层云常参与大范围云系和降水系统的推进，高积云既可只是稳定层附近的波动或对流单体图案，也可在絮状、堡状、混乱天空等形态中提示中层不稳定和天气变化。

## 高层云

高层云是中云族中的大范围云幕，常呈灰白或灰色，覆盖天空较广。它和雨层云、高积云、高层云下方的碎云或降水云系关系密切：高层云继续增厚、降低并伴降水时，可能进入雨层云云系；高层云下方可同时出现高积云或其他较低云层。

## 高积云

高积云是中云族中由小云块、云片、云条组成的云。它常呈白色或灰色，背阳侧可有阴影。《云彩收集者手册》给出一个实用辨析：层积云、高积云、卷积云都可呈团块状，但层积云最低、云块看起来最大且更混乱；卷积云最高，云块很小像颗粒；高积云云块大小介于两者之间。该书还用对流单体解释常见有序团簇：较暖空气在云层下方上升形成云团，较冷空气下沉形成云隙，从混沌中组织出规则图案。

| 识别特征 | 多层、多形态、多方向；可见高积云向卷积云演变，高空密卷云同时变化。 |
| 天气意义 | 表示中高云层结构复杂，可能处在天气系统或多层湿区中；需连续观测演变。 |

## 观测和辨析

| 辨析对象 | 关键区分 |
| --- | --- |
| 高层云 vs 卷层云 | 高层云更灰、更厚，太阳轮廓模糊或不可见，通常不产生典型晕；卷层云更高更薄，常伴晕。 |
| 高层云 vs 雨层云 | 高层云未必降水，结构较均匀；雨层云更厚暗，常伴连续性降水，云底更低。 |
| 高积云 vs 层积云 | 高积云云块视角较小、更整齐，属中云；层积云更低，云块看起来更大且较混乱。 |
| 高积云 vs 卷积云 | 高积云常有阴影和灰暗厚度；卷积云更小、更白、更高，通常无明显阴影。 |
| 波状高积云 vs 辐辏状高积云 | 波状云条多近似平行、与波动有关；辐辏状因透视效应似向地平线汇聚。 |
| 荚状高积云 vs 堡状高积云 | 荚状高积云平滑、透镜状；堡状高积云顶部有炮塔状凸起，指示不稳定。 |
