![《中国云图》PDF 第 13 页](../../../assets/images/clouds/china-cloud-atlas/page-013.webp)

## 定义

云是由大气中水汽凝结或凝华而形成的微小水滴、过冷水滴、冰晶、雪晶等单一或混合组成，形状各异、飘浮在天空中可见的聚合体。

云的生成、外形特征、量的多少、在天空中的分布和演变，可以显示当时大气运动、稳定程度和水汽分布状况，也是未来天气演变的主要征兆之一。客观地观测分析云的宏观演变、描述天气实况，是研究天气变化规律的重要内容之一。

## 分类方法

根据观测和天气预报的需要，按云底距地面的高度，云可分为低云、中云、高云三族；再按云的宏观特征、物理结构和成因划分为十属二十九类云状。
