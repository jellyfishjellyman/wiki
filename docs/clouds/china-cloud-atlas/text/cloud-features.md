# 云的特征

来源：《中国云图》文字说明第 2-6 页，对应 PDF 第 14-18 页。

本页整理原书“云的特征”的主干结构和可确认要点。原页图像保留在 [PDF 第 001-020 页总览](../pages-001-020.md) 中，供逐字复核。

## 总述

云的生成和发展是复杂的物理过程。大气中的温度、湿度、气流、凝结核和冰核数量等因素相互作用，形成绚丽多彩并瞬息多变的云。熟练掌握云的特征，有助于准确识别各种云状，提高观测云的水平。

## 低云

低云包括积云、积雨云、层积云、层云、雨层云五属。低云多由微小水滴组成，厚的或垂直发展旺盛的低云，其下部多由微小水滴组成，中、上部可由微小水滴、过冷水滴和冰晶混合组成。低云云底距地面高度较低，一般低于 2500 米，并随季节、天气条件和地理位置变化。

多数低云可能产生降水。雨层云多出现连续性降水，积雨云多产生阵性降水，有时降水量很大。

### 积云 Cu

积云轮廓分明，顶部凸起，底部平坦，云块之间多不相连。它由低层空气对流作用使水汽凝结，或在冬季凝华而形成，属于直展云。

| 云类 | 代码 | 特征摘要 |
| --- | --- | --- |
| 淡积云 | Cu hum | 发展初期的积云，云体底部较平，个体不大，顶部呈圆弧形凸起，水平宽度大于垂直厚度，晴天多见。 |
| 碎积云 | Fc | 云体很小，零散分布，形状多变，为白色碎块，多为初生或破碎的积云。 |
| 浓积云 | Cu cong | 云体高大，轮廓清晰，底部较平且较阴暗，垂直发展旺盛，顶部常呈花椰菜状。发展很旺盛时，顶部可出现头巾状云。 |

### 积雨云 Cb

积雨云由浓积云演变而成，云体浓厚庞大，垂直发展旺盛，顶部已冰晶化，常有毛丝般纤维结构，并可随发展展平成砧状。它是对流云发展的极盛阶段，常产生较强阵性降水，并伴有大风、雷电等现象，有时出现冰雹，偶有龙卷。

| 云类 | 代码 | 特征摘要 |
| --- | --- | --- |
| 秃积雨云 | Cb calv | 浓积云向鬃积雨云发展的过渡阶段，顶部开始冰晶化，轮廓变模糊，出现少量白色毛状云丝，但尚未扩展开来。 |
| 鬃积雨云 | Cb cap | 积雨云成熟阶段，云顶有白色毛丝般纤维结构，并已扩展成鬃状或铁砧状，云底阴暗而混乱。 |

### 层积云 Sc

层积云云体大小、厚薄不均，形状有条状、片状或团状，呈灰白色或暗灰色。薄的层积云可辨认太阳位置，厚的层积云较阴暗。它可成行或波状排列，也可排列不规则。

层积云厚度在 100-2000 米之间，由直径 5-40 微米水滴组成；冬季和高原地区的层积云可由过冷水滴、冰晶和雪晶组成。层积云一般由大气波动和对流混合作用使水汽凝结而成，有时由局地辐射冷却及湍流混合形成。层积云云底较低，当云层发展较厚时常出现短时降雨，冬季可降雪。

| 云类 | 代码 | 特征摘要 |
| --- | --- | --- |
| 透光层积云 | Sc tra | 云体较薄，呈灰白色，排列比较整齐，边缘较明亮。云体之间有明显缝隙，可分辨出日月位置；如果层积云上边还有云层，也能看到。 |
| 蔽光层积云 | Sc op | 云块较密集、较厚，呈暗灰色，云块之间无缝隙，可以遮蔽日月；云底有明显波状起伏，常布满天空，有时产生降水。 |
| 积云性层积云 | Sc cug | 云体呈扁平长条形，灰白色或暗灰色，顶部具有积云特征。它由衰退的积云或积雨云扩展、平衍而成；有时由傍晚地面散热、空气抬升、水汽凝结形成。其出现显示对流减弱并趋向稳定，有时降零星小雨。 |
| 堡状层积云 | Sc cast | 云体细长，底部较平，顶部凸起一个或几个云堡，但高度不同，有继续发展的趋势；视角宽度大于 5°，远看好像城堡或长条形锯齿。若水汽条件较好且垂直气流继续增强，有利于积雨云发展，预示当地将有雷阵雨天气。 |
| 荚状层积云 | Sc lent | 云体呈豆荚状或梭子形，中间较厚、边缘较薄，在地形影响气流形成驻波的作用下形成；云体视角宽度为 5°-30°。 |

### 层云 St

层云通常呈灰色或灰白色，似浓雾，云底较低但不接地，经常笼罩山体和高层建筑。层云由直径 5-30 微米的水滴或过冷水滴组成，厚度一般在 400-500 米之间。

层云多在大气稳定条件下，由夜间辐射冷却或乱流混合作用使水汽凝结而成，也可由雾抬升形成。日出后气温逐渐升高，稳定层被破坏，层云也逐渐消散。层云有时降毛毛雨，冬季降米雪。

| 云类 | 代码 | 特征摘要 |
| --- | --- | --- |
| 碎层云 | Fs | 层云在逐渐消散过程中，或辐射雾因扰动抬升而形成碎层云；云体呈灰色或灰白色，支离破碎，形状多变，出现时多预示晴天。 |

### 雨层云 Ns

雨层云云底很低，云层很厚，一般厚度为 4000-5000 米，能遮蔽日月，呈暗灰色，云底经常出现碎雨云。雨层云覆盖范围很大，常布满天空。云层中下部由水滴和过冷水滴组成；北方和高原地区的雨层云中上部由过冷水滴、冰晶和雪晶组成。

雨层云常出现在暖锋云系中，有时也出现在其他天气系统中。它由暖湿空气沿锋面坡缓慢滑升、绝热冷却而形成，常产生连续性降雨；北方冬季和高原地区夏季均会降雪。农谚“天上灰布悬，雨丝定连绵”多指雨层云的连续性降水。

| 云类 | 代码 | 特征摘要 |
| --- | --- | --- |
| 碎雨云 | Fn | 云底很低，通常只有 50-400 米，云体零散破碎、形状多变、移动较快，呈灰色或暗灰色；经常出现在雨层云、积雨云或较厚高层云云底下边，是降水时空气湿度增大、在乱流作用下水汽凝结形成的。 |

## 中云

中云包括高层云和高积云两属。中云由微小水滴、过冷水滴，或与冰晶、雪晶混合组成。云底高度一般在 2500-5000 米。

### 高层云 As

高层云是灰色或灰白色的云幕，云层较厚，多在 1500-3500 米之间，云底部常出现条纹结构，可部分或全部布满天空。

| 云类 | 代码 | 特征摘要 |
| --- | --- | --- |
| 透光高层云 | As tra | 云层较薄，厚度较均匀，透过云层可见模糊的日月轮廓。 |
| 蔽光高层云 | As op | 云层较厚，厚度变化较大，云底灰色或深灰色，地面不能辨认太阳或月亮。 |

### 高积云 Ac

高积云云体较小，个体分明，厚薄和形状不同。常呈扁圆形、瓦块状、鱼鳞片或水波状密集云条，常成行或波状排列。

高积云由微小水滴或过冷水滴与冰晶混合组成。每当日月光透过薄的高积云时，常因高积云中的微小水滴或冰晶对光的衍射而形成内蓝外红的光环，称为华。

在高空逆温层上下，空气密度和风都有较大差异，逆温层附近经常产生波动。若逆温层下聚集较多水汽，在波峰处因空气上升冷却而形成高积云。云体不厚、比较稳定、很少变化时预示晴天。若高积云厚度继续增厚并逐渐融合成层，则显示天气将有变化，甚至会出现降水。

| 云类 | 代码 | 特征摘要 |
| --- | --- | --- |
| 透光高积云 | Ac tra | 云体较薄，呈白色，在天空中整齐排列，云体之间有缝隙，可见蓝天；有时云体之间虽无缝隙，但边缘较明亮，透过云体边缘可分辨日月位置。 |
| 蔽光高积云 | Ac op | 云体较厚，呈暗灰色，云体已融合成层，日月不能辨认，有时会出现微量降水。 |
| 荚状高积云 | Ac lent | 云体中间厚、边缘薄，中间呈暗灰色，边缘呈白色，轮廓分明，一般呈豆荚状或椭圆形，孤立分散在天空；遮挡日月光线时可出现虹彩。它由地形影响气流形成的驻波作用生成，多出现在晴朗有风的天气。 |
| 积云性高积云 | Ac cug | 云块有大有小，呈灰白色，中间稍厚，顶部略有凸起特征。由衰退的积云或积雨云扩展演变而成，出现时预示天气逐渐趋于稳定。 |
| 絮状高积云 | Ac flo | 云块大小不一，带有积状云外形，云团下部较破碎，很像破碎棉絮团，分散在天空，高度不相同，呈灰白色或灰色，可出现雪幡。它由高空潮湿气层很不稳定、有强湍流混合作用形成；有的地区认为其出现预示雷雨天气。 |
| 堡状高积云 | Ac cast | 高积云呈水平条状分布在高空，顶部有多处向上凸起，很像城堡，也有的呈锯齿状；出现时预示将有不稳定的雷阵雨天气。 |

## 高云

高云包括卷云、卷层云、卷积云三属。高云由微小冰晶组成，云底高度一般在 5000 米以上，高原地区可较低。

### 卷云 Ci

卷云由冰晶组成，有毛丝般光泽，常见丝条状、片状、羽毛状、钩状、团状、砧状等形态。

| 云类 | 代码 | 特征摘要 |
| --- | --- | --- |
| 毛卷云 | Ci fil | 云片较薄，颜色洁白，毛丝般纤维结构清晰，受高空风影响，云丝分散，形状多样，像乱丝、羽毛、马尾等；日月透过，地物阴影较明显。毛卷云出现时多预示晴天；若演变中厚度和云量增加并发展成卷层云，则预示天气将有变化。 |
| 密卷云 | Ci dens | 云体中部较厚，边缘薄的部分呈白色，毛丝般结构仍较明显。云丝密集、聚合成片，云量逐渐增多时，透过密卷云可观测到不完整的晕。密卷云一般显示天气较稳定；若继续发展并演变成卷层云，则预示未来天气将有变化。 |
| 伪卷云 | Ci not | 云体较大且厚密，一般呈砧状，由积雨云衰退时云砧脱离主体演变而成；通常在积雨云逐渐消散时可观测到。 |
| 钩卷云 | Ci unc | 云体很薄，呈白色，云丝往往平行排列，有时倾斜下垂，向上的一头有小钩或小簇，像逗点符号。钩卷云常分散在天空；若系统地移入天空并继续发展，预示可能有不稳定天气系统影响测站并出现阴雨天气。 |

### 卷层云 Cs

卷层云云层较均匀，常呈乳白色。日月透过云层时轮廓清楚，并经常有晕圈出现，地面物体有影。

卷层云逐渐增厚、高度降低并继续发展时，预示将有天气系统影响测站。原书引用民间谚语“日晕三更雨，月晕午时风”说明这种指示意义。反之，如果卷层云无明显变化，云量还逐渐减少，未来天气不会有大的变化。

| 云类 | 代码 | 特征摘要 |
| --- | --- | --- |
| 毛卷层云 | Cs fil | 云层厚薄不均，云底也不平整，毛丝般纤维结构比较明显，云的顶部比较平坦，略有微小起伏。 |
| 薄幕卷层云 | Cs nebu | 云层薄而均匀，似薄幕，毛丝般结构不明显，有时易误认为无云。云层由冰晶组成，每当日月光透过云层时，会出现晕的现象。 |

### 卷积云 Cc

卷积云云块很小，呈白色鱼鳞片状，成行、成群排列在高空，有时像微风吹拂水面形成的小波纹。卷积云由高空大气层结不稳定产生波动而形成。

如果天空云的分布以卷积云为主，又伴有卷云、卷层云并系统发展，通常预示将有不稳定的天气系统影响测站，常出现阴雨、大风天气。原书引用农谚“鱼鳞天，不雨也风颠”说明这种情况。
