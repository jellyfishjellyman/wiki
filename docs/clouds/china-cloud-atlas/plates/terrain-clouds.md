## 珠峰旗云与山地对流云

### 图 246：珠穆朗玛峰旗云

| 拍摄时间 | 1979年10月 |
| 拍摄方向 | S |
| 拍摄者 | 曾曙生 |
| 原分页 | [PDF 第 275 页](../pages-261-280.md) |

珠穆朗玛峰的东南侧有云层沿山坡向上伸展，云顶受西风带气流影响，起伏不平，中间波峰比较明显，两边比较平滑。

## 层积云和浓积云

### 图 249：珠穆朗玛峰的层积云

![图 249：珠穆朗玛峰的层积云](../../../assets/images/clouds/china-cloud-atlas/page-278.webp)

| 字段 | 内容 |
| --- | --- |
| 拍摄地点 | 西藏 珠峰北侧 |
| 拍摄时间 | 1981年9月 |
| 拍摄方向 | SW |
| 拍摄者 | 王富州 |
| 原分页 | [PDF 第 278 页](../pages-261-280.md) |

图中是珠峰北坡出现的层积云。由于有一逆温层，层积云呈条状、带状，越过山脊时受地形影响，产生波动，部分云裂开，整体变化不大。

## 冰川云、瀑布云和云海

### 图 252：冰川上的淡积云

| 拍摄方向 | N |

图中是冰川附近形成的淡积云。这些淡积云是由于冰川表面融化，水汽蒸发后在空中凝结而形成的。

### 图 256：碎积云

![图 256：碎积云](../../../assets/images/clouds/china-cloud-atlas/page-285.webp)

| 字段 | 内容 |
| --- | --- |
| 拍摄地点 | 台湾 阿里山 |
| 拍摄时间 | 2002年5月18日 |
| 拍摄方向 | 原页未列出 |
| 拍摄者 | 张蓄 |
| 原分页 | [PDF 第 285 页](../pages-281-300.md) |

阿里山水汽丰沛，因局地受热增温，水汽凝结成碎云。碎云云块随山地气流抬升不断地升入空中，云块边缘破碎，呈灰白色。
