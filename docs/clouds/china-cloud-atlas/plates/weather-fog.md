## 湖区浓雾、辐射雾与浓雾

### 图 196：湖区浓雾

![图 196：湖区浓雾](../../../assets/images/clouds/china-cloud-atlas/page-221.webp)

| 字段 | 内容 |
| --- | --- |
| 拍摄地点 | 吉林 长春 |
| 拍摄时间 | 1988年7月13日06时40分 |
| 拍摄方向 | E |
| 拍摄者 | 郭恩铭 |
| 原分页 | [PDF 第 221 页](../pages-221-240.md) |

早晨长春市南湖水面空气达到饱和状态，由水汽凝结而形成湖区浓雾。

### 图 197：辐射雾

![图 197：辐射雾](../../../assets/images/clouds/china-cloud-atlas/page-222.webp)

| 字段 | 内容 |
| --- | --- |
| 拍摄地点 | 辽宁 锦西 |
| 拍摄时间 | 1987年7月23日06时15分 |
| 拍摄方向 | W |
| 拍摄者 | 郭恩铭 |
| 原分页 | [PDF 第 222 页](../pages-221-240.md) |

图中的村庄西边有一条小河，河的两岸近地面层水汽较多，每当夜间辐射冷却，气温降低时，水汽凝结就形成了几十米厚的雾层，即辐射雾。

### 图 198：辐射雾

![图 198：辐射雾](../../../assets/images/clouds/china-cloud-atlas/page-223.webp)

| 字段 | 内容 |
| --- | --- |
| 拍摄地点 | 四川 成都双流机场 |
| 拍摄时间 | 1986年12月2日09时17分 |
| 拍摄方向 | SW |
| 拍摄者 | 郭恩铭 |
| 原分页 | [PDF 第 223 页](../pages-221-240.md) |

早晨 04 时 30 分辐射雾已形成，在雾维持过程中有时变浓，有时减弱，一直持续到 09 时 30 分雾才逐渐消散。拍摄时能见度小于 150 米。

### 图 199：浓雾

![图 199：浓雾](../../../assets/images/clouds/china-cloud-atlas/page-224.webp)

| 字段 | 内容 |
| --- | --- |

| 拍摄方向 | N |
| 拍摄者 | 郭恩铭 |

春雨过后，湿度较大，大气层结比较稳定，夜间辐射冷却降温，水汽凝结形成雾。图中是早晨 07 时的浓雾，能见度小于 100 米。

## 锋面雾与海雾

### 图 201：锋面雾

![图 201：锋面雾](../../../assets/images/clouds/china-cloud-atlas/page-226.webp)

| 字段 | 内容 |
| --- | --- |
| 拍摄地点 | 湖北 长江江面 |
| 拍摄时间 | 1989年10月20日07时10分 |
| 拍摄方向 | SE |
| 拍摄者 | 郭恩铭 |
| 原分页 | [PDF 第 226 页](../pages-221-240.md) |

夜间长江江面开始降雨，凌晨雨停。冷空气移至较暖的江面上，水汽凝结而形成锋面雾。图中是武昌西部江面上的锋面雾。

### 图 202：锋面雾

![图 202：锋面雾](../../../assets/images/clouds/china-cloud-atlas/page-227.webp)

| 字段 | 内容 |
| --- | --- |
| 拍摄地点 | 上海 黄浦江 |
| 拍摄时间 | 1989年1月6日22时 |
| 拍摄方向 | E |
| 拍摄者 | 郭恩铭 |
| 原分页 | [PDF 第 227 页](../pages-221-240.md) |

上海当天处于暖区，降小雨，空气中湿度增大，16 时降雨停止，18 时降温后黄浦江面形成锋面雾，能见度 100-250 米。

### 图 203：海雾（平流雾）

![图 203：海雾（平流雾）](../../../assets/images/clouds/china-cloud-atlas/page-228.webp)

![图 203：海雾（平流雾）](../../../assets/images/clouds/china-cloud-atlas/page-229.webp)

| 字段 | 内容 |
| --- | --- |
| 拍摄地点 | 福建 鼓浪屿 |
| 拍摄时间 | 1987年6月4日09时30分 |
| 拍摄方向 | SE |
| 拍摄者 | 郭恩铭 |
| 原分页 | [PDF 第 228-229 页](../pages-221-240.md) |

海上的平流雾从鼓浪屿南边海上移来，逐渐向岸边延伸，岸边一块巨石只见顶部，海雾又覆盖了岸边。海雾在岸边持续了近 40 分钟后，逐渐往南退移而后消失。

### 图 204：蒸发雾

![图 204：蒸发雾](../../../assets/images/clouds/china-cloud-atlas/page-229.webp)

| 字段 | 内容 |
| --- | --- |
| 拍摄地点 | 北京 紫竹院 |

| 拍摄者 | 郭恩铭 |

夜间降雪已停止，早晨近地面气温较低，河面温度较高，水汽蒸发后在水面上凝结成蒸发雾。

### 图 205：轻雾

![图 205：轻雾](../../../assets/images/clouds/china-cloud-atlas/page-230.webp)

| 字段 | 内容 |
| --- | --- |
| 拍摄地点 | 辽宁 虹螺山 |
| 拍摄时间 | 1987年7月22日06时05分 |
| 拍摄方向 | NE |
| 拍摄者 | 郭恩铭 |
| 原分页 | [PDF 第 230 页](../pages-221-240.md) |

早晨，山地园林地区日出后，受阳光照射，在近地面有几米厚的灰白色轻雾层。轻雾是夜间地面长波辐射很强、气温降低、水汽凝结而形成的。
