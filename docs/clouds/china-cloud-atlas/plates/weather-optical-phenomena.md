## 宝光

### 图 171：宝光

![图 171：宝光](../../../assets/images/clouds/china-cloud-atlas/page-199.webp)

| 字段 | 内容 |
| --- | --- |
| 拍摄地点 | 安徽 黄山 |
| 拍摄时间 | 2002年2月23日11时06分 |
| 拍摄方向 | ENE |
| 拍摄者 | 胡福生 |
| 原分页 | [PDF 第 199 页](../pages-181-200.md) |

宝光是一种大气光学现象。当人面对云雾而立，阳光从人的背后照射过来，将人的影子投射于云雾上，由于阳光经云雾水滴的衍射，在人的影子周围产生了内紫外红的光环即是宝光，俗称峨眉宝光。

| 拍摄者 | 赖比星 |

高山上云雾弥漫，阳光从观测者背后照射时出现的宝光。

### 图 174：虹彩

![图 174：虹彩](../../../assets/images/clouds/china-cloud-atlas/page-202.webp)

| 字段 | 内容 |
| --- | --- |
| 拍摄地点 | 北京 西郊 |
| 拍摄时间 | 1985年9月2日13时50分 |
| 拍摄方向 | SW |
| 拍摄者 | 郭恩铭 |
| 原分页 | [PDF 第 202 页](../pages-201-220.md) |

中午过后天空出现了卷云和卷积云，云层不厚有些发毛，它是由过冷水滴和冰晶组成。当太阳直射在卷积云云顶时，由于过冷水滴和冰晶对阳光的衍射作用，形成非常美丽的虹彩。

### 图 175：曙暮光弧

![图 175：曙暮光弧](../../../assets/images/clouds/china-cloud-atlas/page-203.webp)

| 字段 | 内容 |
| --- | --- |
| 拍摄地点 | 四川 上空 |
| 拍摄时间 | 1987年9月16日19时40分 |
| 拍摄方向 | W |
| 拍摄者 | 郭恩铭 |
| 原分页 | [PDF 第 203 页](../pages-201-220.md) |

日落西山，天空中分布着大片高积云。飞机航行在10000米高空观测到西方有砧状积雨云。高积云已呈暗黑色，天空是深蓝色，两者之间有黄红色亮带即曙暮光弧。这是大气中粒子对阳光散射的结果。

### 图 176：蜃景

![图 176：蜃景](../../../assets/images/clouds/china-cloud-atlas/page-204.webp)

| 字段 | 内容 |
| --- | --- |
| 拍摄地点 | 山东 蓬莱 |
| 拍摄时间 | 1988年6月17日14时25分 |
| 拍摄方向 | ESE |
| 拍摄者 | 孙玉平 |
| 原分页 | [PDF 第 204 页](../pages-201-220.md) |

蜃景是海面上由于大气折射、反射而形成的光学现象。天气晴好海面受热后低层空气上下密度不同，太阳光穿过空气层产生折射和全反射，海面上出现似有海岛、船舶的蜃景。有时还会呈现城楼的蜃景。
