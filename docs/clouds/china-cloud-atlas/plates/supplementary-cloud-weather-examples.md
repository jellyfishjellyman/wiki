## 极地和海外云例

### 图 260：北极上空的高积云（宝光）

![图 260：北极上空的高积云（宝光）](../../../assets/images/clouds/china-cloud-atlas/page-291.webp)

| 字段 | 内容 |
| --- | --- |
| 拍摄地点 | 北极上空 |
| 航行高度 | 约 9000 米 |
| 拍摄时间 | 2001年8月30日 |
| 拍摄方向 | 原页未列出 |
| 拍摄者 | 郭恩铭 |
| 原分页 | [PDF 第 291 页](../pages-281-300.md) |

图中是飞机飞越北极上空，航行高度约 9000 米，从飞机上观测到的极地上空的高积云。云顶呈波状，起伏不平。当阳光映照飞机时，飞机的影子投射在高积云上，在它的周围出现了宝光。
