## 枝状闪电

### 图 180：枝状闪电

![图 180：枝状闪电](../../../assets/images/clouds/china-cloud-atlas/page-208.webp)

| 字段 | 内容 |
| --- | --- |
| 拍摄地点 | 北京 海淀 |
| 拍摄时间 | 1987年7月30日21时50分 |
| 拍摄方向 | SW |
| 拍摄者 | 郭恩铭 |
| 原分页 | [PDF 第 208 页](../pages-201-220.md) |

积雨云经过测站向正南方向移去，云体庞大，云底放电很猛烈。测站南面和西南方向频繁出现枝状闪电。图中右侧的闪电最强，出现了强光通道，另外两个发展不太猛烈。
