# 云知识 Wiki

本区按知识主题整理云的概念、分类、识别、形成机制、天气意义和观测方法。现有《中国云图》整理页、扫描页和 OCR 页作为资料库保留，用于校订图版、追溯来源和补充案例；主线不再按单书页码或原书目录组织。

## 学习路径

1. 从 [云的基础概念](basics.md) 理解云的组成、形成条件和观测价值。
2. 阅读 [云的分类体系](classification.md)，建立低云、中云、高云和十属二十九类云状的框架。
3. 按云族进入 [低云](low-clouds.md)、[中云](mid-clouds.md)、[高云](high-clouds.md)，再专题查看 [积云与积雨云](cumulus-cumulonimbus.md)、[层状云](layered-clouds.md) 和 [特殊云形](special-forms.md)。
4. 用 [云图图例库](gallery.md) 对照图片练习识别，并在 [术语表](glossary.md) 中查找缩写和概念。
