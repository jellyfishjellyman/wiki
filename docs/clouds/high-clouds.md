## 定义和范围

高云通常位于 5000 米以上，高原地区可相对较低。《云彩收集者手册》译本常把高云族云底写作约 5000-13500 米。本 Wiki 以《中国云图》的“三族、十属、二十九类云状”为主，保留不同资料高度阈值差异。

高云多由冰晶组成，常呈白色、纤维状、薄幕状或细小波纹状。它们与天气系统和光象关系密切：卷云和卷层云系统性移入、增厚、降低，可能是天气系统接近的早期线索；卷层云中的六角冰晶可形成 22 度晕、幻日、日柱等晕现象；卷积云常短暂、细小，常由密卷云或高层冰晶云演变而成。

## 卷云

《云彩收集者手册》把卷云描述为高空冰晶下落并穿过对流层上部高空风而形成的云条，外观像白色头发。卷云可孤立出现，也可随着高空湿度增加逐渐增厚并铺展；若这种过程持续发展，可能进入卷层云、高层云、雨层云的云系序列，并在一段时间后带来降水。这个判断必须结合云系演变，不应把单条卷云直接等同于降水预报。

## 卷层云

卷层云是高空冰晶云幕，常薄而广，能覆盖大面积天空。它可以呈纤维状，也可以均匀如薄幕。《云彩收集者手册》强调卷层云在产生晕现象的高云中表现突出：阳光或月光被冰晶折射、反射，可形成光环、光弧、光斑。是否出现晕还取决于冰晶形状、取向和云层位置；没有晕不代表一定不是卷层云。

## 卷积云

卷积云由非常小的高空云块或波纹组成，常呈白色、鱼鳞片状或细小波状。《云彩收集者手册》指出卷积云距离地面很远，通常在约 9500 米高空，云块看起来很小；若高于地平线 30 度以上，伸直手臂观察，小云块通常不大于一根手指宽。该方法可帮助区分卷积云和较低、云块较大的高积云。卷积云罕见且短暂，很多“鱼鳞状云”实际可能是高积云，需谨慎判断。

## 观测和辨析

| 辨析对象 | 关键区分 |
| --- | --- |
| 卷云 vs 卷层云 | 卷云是丝缕、羽毛、钩状或片状个体；卷层云是扩展成幕的冰晶云层。 |
| 密卷云 vs 高积云 | 密卷云边缘有纤维状冰晶结构，高积云多为小云块且常有阴影。 |
| 卷层云 vs 高层云 | 卷层云更薄、更高，常有晕；高层云更灰、更厚，太阳像隔毛玻璃或被遮蔽。 |
| 卷积云 vs 高积云 | 卷积云云块更小、更白、通常无明显阴影；高积云云块较大、有灰暗厚度。 |
| 伪卷云 vs 普通卷云 | 伪卷云需要结合积雨云云砧脱离、雷阵雨后或云砧残余的生成史。 |
| 晕 vs 华 | 晕多由卷层云冰晶折射/反射形成，22 度角常见；华由小水滴或冰晶衍射形成，靠近日月且色环较小。 |
