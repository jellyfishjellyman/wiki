# 术语表

| 术语 | 说明 |
| --- | --- |
| 云族 | 按云底高度划分的最高层级，常用低云、中云、高云。 |
| 云属 | 云分类的主要形态层级，如积云、层积云、高积云、卷云。 |
| 云类 | 云属下更细的形态或结构类别，如淡积云、荚状高积云。 |
| 云底 | 云体可见底部，观测高度分类时常以云底为依据。 |
| 云量 | 云遮蔽天空的比例，需结合观测规范记录。 |
| 透光 | 云层较薄，可见蓝天或辨认日月位置。 |
| 蔽光 | 云层较厚，遮蔽日月，天空较暗。 |
| 对流 | 空气垂直运动，积云和积雨云发展的核心过程。 |
| 凝结 | 水汽转变为液态水滴。 |
| 凝华 | 水汽直接转变为冰晶。 |
| 过冷水滴 | 温度低于 0 摄氏度但仍保持液态的水滴。 |
| 雨幡 | 降水从云中下落形成的帘状或丝状结构，可未到达地面。 |
| 雪幡 | 固态降水粒子从云中下落形成的幡状结构。 |
| 云砧 | 积雨云顶部在高空风作用下水平伸展形成的铁砧状结构。 |
| 晕 | 日月光经卷层云中冰晶折射和反射形成的光环。 |
| 华 | 日月光经薄云中小水滴或冰晶衍射形成的彩色光环。 |
| 宝光 | 观测者影子投射到云雾上并伴随彩色光环的现象。 |
| CL、CM、CH | 《中国云图》图版中使用的低云、中云、高云编码前缀。具体编码需按原图版校订。 |
