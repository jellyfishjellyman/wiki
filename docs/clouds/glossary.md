| 线状闪电 | 积雨云中形成的线状放电通道，可为云地闪电。 |
| 枝状闪电 | 带有分支的闪电通道，常见于强积雨云放电。 |
| 球状闪电 | 稀有球形发光放电现象，《中国云图》图 183 记录于频繁闪电之后；机制解释需谨慎。 |
| 云中闪电 | 闪电发生在云体内部或云间，不直接接地。 |
| 龙卷 | 强烈旋转风暴从积雨云底部向下伸展形成的漏斗状或柱状旋涡；是否接地需单独记录。 |
| 尘卷风 | 局地强烈受热使小股空气旋转上升并卷起尘沙形成的旋风。 |
| 沙尘暴 | 强风卷起大量沙尘造成天空土黄、能见度明显下降的天气现象。 |
| 晕 | 日月光经卷层云中冰晶折射和反射形成的光环。 |
| 华 | 日月光经薄云中小水滴或冰晶衍射形成的彩色光环。 |
| 假日 | 卷层云冰晶使太阳两侧晕圈上出现亮斑；与“幻日”术语关系待校订。 |
| 幻日 | `sundog` / `parhelion` 的常见中文译名，可能对应《中国云图》“假日”，待校订。 |
| 日柱 | 太阳光透过卷层云或冰晶层后形成的垂直光柱。 |
| 宝光 | 观测者影子投射到云雾上并伴随彩色光环的现象。 |
| 布罗肯幽灵 | 观测者影子投射到云雾上并伴随宝光的山地 / 云雾光象；《一天一朵云》索引含 `Brocken spectres`。 |
| 虹 | 阳光经降水水滴折射和反射形成的彩色弧，色带通常内紫外红。 |
| 霓 | 虹外侧较暗的副虹，色带顺序常与虹相反，即内红外紫。 |
| 雾虹 | 小水滴雾幕中形成的浅色虹；《一天一朵云》索引含 `fogbows`，与云虹、虹、霓差异待校订。 |
| 云虹 | 小水滴云幕中形成的淡色虹；《云彩收集者手册》《一天一朵云》均有 EPUB 线索，逐图待校订。 |
| 虹彩 | 云滴或冰晶衍射使云边、云顶出现彩色的光象。 |
| 曙暮光弧 | 日出或日落附近大气粒子散射形成的黄红色亮带，《中国云图》图 175 为高空观测个例。 |
| 曙暮光条 | 低太阳高度时被云或地形遮挡后形成的明暗相间光束；《一天一朵云》索引含 `crepuscular rays`。 |
| 反曙暮光条 | 与曙暮光条在天空反太阳方向汇聚的光束；《一天一朵云》索引含 `anti-crepuscular rays`。 |
| 彩虹轮 | 彩虹与反曙暮光条组合出现时形成的轮辐状视觉效果；《一天一朵云》个例待校订。 |
| 环天顶弧 | 高空冰晶造成的晕族光象，通常位于太阳上方较高处；《一天一朵云》索引含 `circumzenithal arcs`。 |
| 外接晕 | 晕族光象之一；《一天一朵云》索引含 `circumscribed halo`，中文名和几何机制待校订。 |
| 月晕 | 月光经冰晶云层形成的晕族光象；22 度月晕个例见《一天一朵云》待校订案例。 |
| 对日效应 | 《一天一朵云》索引中的 `anti-solar effects`，可能包含宝光、布罗肯幽灵等反日点附近光象，范围待校订。 |
| 蜃景 | 低层空气密度差异导致光线折射或全反射，在海面等处形成似有远物的光学现象。 |
| CL、CM、CH | 《中国云图》图版中使用的低云、中云、高云编码前缀。具体编码需按原图版校订。 |
