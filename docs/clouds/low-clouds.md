## 定义和范围

低云是云底较低的一组云，通常位于低层天空。《中国云图》按观测和天气预报需要，把云按云底高度分为低云、中云、高云三族，再按宏观特征、物理结构和成因划分为十属二十九类云状。低云包括五个云属：积云、积雨云、层积云、层云、雨层云。

低云多由微小水滴组成；在较厚或垂直发展旺盛的云体内，中上部可含过冷水滴、冰晶、雪晶，甚至形成混合云结构。积雨云虽然云底属低云，但成熟时可贯穿低、中、高多个高度层，因此它在天气意义上不能只按“低”理解。

## 低云的共同识别维度

| 维度 | 观察问题 | 对低云分类的意义 |
| --- | --- | --- |
| 云底高度 | 云底是否很低，是否接近山坡、海面或地物？ | 区分低云与中云、高云；层云与雾还需看是否接地。 |
| 云体结构 | 是单体凸起、片状团块，还是均匀云层？ | 单体凸起偏积云/积雨云；片状团块偏层积云；均匀低层偏层云或雨层云。 |
| 垂直发展 | 云顶是否快速增高，是否花椰菜状或冰晶化？ | 判断淡积云、浓积云、积雨云阶段。 |
| 透光性 | 能否见太阳位置或蓝天缝隙？ | 透光与蔽光层积云、高层云、雨层云辨析的重要依据。 |
| 降水结构 | 是否有雨幡、降水带、碎雨云、泄雹带？ | 指向积雨云、雨层云或降水云底附属结构。 |
| 运动和演变 | 云体移动方向、增厚速度、是否合并或破碎？ | 对流发展和天气预警比单张照片更可靠。 |
| 环境背景 | 地形、海陆风、锋面、低压、日变化是否明显？ | 解释同一云类的不同形态和天气意义。 |

## 天气意义

| 云类 / 结构 | 天气意义 | 观测提示 |
| --- | --- | --- |
| 淡积云 | 多见于晴好天气或弱对流环境 | 若只在白天发展、傍晚消散，通常天气较稳定。 |
| 碎积云 | 可为初生积云，也可为积云破碎 | 看是否逐渐长大成淡积云，或是否贴近降水云底。 |
| 浓积云 | 对流增强，可能转为积雨云 | 重点看云顶增高、云底变暗、雨幡和帽状云。 |
| 秃积雨云 | 已进入积雨云阶段，降水风险升高 | 云顶冰晶化和云底黑灰是关键。 |
| 鬃积雨云 / 云砧 | 成熟强对流，常伴雷雨、冰雹、大风 | 记录云砧伸展方向、降水带、雷电、移动路径。 |
| 悬球状云底 | 强对流云底或云砧底部的下垂囊袋状结构 | 常提示云内下沉、上升气流扰动；需结合雷雨系统判断。 |
| 层积云 | 可为稳定低云，也可提示低层波动或积云衰退 | 透光性、云块大小、是否堡状或荚状很重要。 |
| 层云 | 可伴毛毛雨，常与雾、低层湿冷或地形有关 | 看云是否接地；接地通常按雾处理。 |
| 雨层云 | 连续性降水的重要云类 | 常为厚暗云层，结构单调，云底可能有碎雨云。 |
