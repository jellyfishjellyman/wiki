# 低云

## 定义

低云是云底较低的一组云，通常云底低于 2500 米。它们多由微小水滴组成；较厚或垂直发展旺盛的低云，中上部可含过冷水滴、冰晶或雪晶。

## 分类

低云包括五属：

| 云属 | 典型云类 | 识别关键词 |
| --- | --- | --- |
| 积云 Cu | 淡积云、碎积云、浓积云 | 云底较平，顶部凸起，轮廓清楚 |
| 积雨云 Cb | 秃积雨云、鬃积雨云 | 云体庞大，垂直发展，常伴雷阵雨 |
| 层积云 Sc | 透光、蔽光、积云性、堡状、荚状层积云 | 低空片状、团状或条状，常成行或波状 |
| 层云 St | 层云、碎层云 | 低而均匀，似雾但不接地 |
| 雨层云 Ns | 雨层云、碎雨云 | 厚而暗，常带连续性降水 |

## 识别特征

低云常占据观测者视野的低空部分，云体尺度较大，阴影和云底结构明显。积状低云边缘清楚，层状低云多呈灰色或暗灰色云层。

## 形成机制

低云形成机制多样：积云和积雨云主要由低层对流发展；层积云常与大气波动、湍流混合或衰退积云扩展有关；层云常由辐射冷却、乱流混合或雾抬升形成；雨层云多见于锋面或大范围抬升条件。

## 天气意义

- 淡积云常对应较稳定的晴好天气。
- 浓积云继续发展，可能转化为积雨云。
- 积雨云常带来阵雨、雷电、大风、冰雹，偶有龙卷。
- 雨层云多对应连续性降水。
- 层云有时降毛毛雨，日出后常随加热而消散。
