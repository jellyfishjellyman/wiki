## 三本书的分工

### 《中国云图》

《中国云图》是当前 Wiki 的事实骨架。它的优势是字段完整、图版明确、观测规范强：

- 有三族、十属、二十九类云状的分类表。
- 有 `CL/CM/CH` 低云、中云、高云编码表。
- 大量图版保留图号、云类代码、PDF 页、地点、时间、拍摄方向、拍摄者和说明文字。
- 包含低云、中云、高云、天气现象、飞机上观测云、地形云和补充个例。

使用方式：

| 内容类型 | 放置位置 | 说明 |
| --- | --- | --- |
| 原始扫描页、OCR、页码回查 | `docs/clouds/china-cloud-atlas/pages-*.md` | 保留原始证据，不作为主题阅读主线 |
| 图版逐条字段 | `docs/clouds/china-cloud-atlas/plates/*.md`、`plate-catalog.md` | 图号、地点、时间、方向、摄影者等字段以这里为准 |
| 主题化知识 | `docs/clouds/*.md` | 由图版归纳识别要点、天气意义、易混淆对象 |
| 典型图片 | 主题页中少量引用 | 用于说明识别要点，不复制整本扫描图集 |

## 资料进入 Wiki 的规则

| 资料类型 | 进入主题页 | 进入图版库 / 档案 | 标注要求 |
| --- | --- | --- | --- |
| 云类定义 | 是 | 可链接原文字说明 | 来源、术语差异 |
| 识别特征 | 是 | 图版页保留原图注 | 若为归纳，说明依据 |
| 图号、地点、时间、方向、拍摄者 | 主题页可精选，图版页必须完整 | 是 | 未核对不得写确定字段 |
| 原书长图注 | 主题页只摘要 | 图版页可较完整转写 | 避免把主题页变成扫描复刻 |
| 天气意义 | 是 | 图版页保留原说明 | 区分直接观测、经验指示、推断 |
| EPUB 现代术语 | 是 | 后续建立 EPUB 术语索引 | 标注待校订 |
| EPUB 图片个例 | 只在校订后进入 | 后续建立 EPUB 案例库 | 未校订不得写具体摄影者/页码 |
| OCR 文本 | 只作线索 | 原分页保留 | 关键字段需对照原图 |

## 校订优先级

1. 图号、云类代码、地点、时间、拍摄方向、拍摄者。
2. 原图注中的形成机制、天气背景、移动方向、演变过程。
3. 云类缩写、拉丁名、中文名和现代术语对应。
4. 天气意义和观测经验。
5. EPUB 图片个例的页码、章节、摄影者、会员编号或日期。
6. 主题页之间的交叉链接和索引入口。

## 字段可信度

| 标记 | 使用条件 | 示例 |
| --- | --- | --- |
| 已按原页图像校订 | 已对照 PDF 页图像或结构化图版页 | 图 159 的地点、时间、方向、拍摄者 |
| 已按结构化档案整理 | 来自 `china-cloud-atlas/plates` 或 `text` 已整理页面 | 云类表、云码表 |
| OCR 待复核 | 仅来自分页 OCR，未看原图 | 目录页错行、疑似错字 |
| EPUB 待校订 | 来自 EPUB 搜索或概念抽取，未建逐图索引 | `pileus` 个例、糙面云个例 |
| 术语待校订 | 中英文/拉丁名/中文译名未对齐 | 假日 vs 幻日，帽状云 vs 幞状云 |
| 原页未列出 | 原书确实未给出该字段 | 图 216 拍摄者原页未列出 |

## 主题页和档案页边界

主题页回答“这个概念是什么、如何识别、有什么天气意义、有哪些代表图版”。档案页回答“原书这一图、这一页、这一字段是什么”。二者互补：

| 页面类型 | 应该包含 | 不应该变成 |
| --- | --- | --- |
| 主题页 | 综合定义、识别表、机制、天气意义、易混淆对象、精选图版 | 单书目录复刻、逐页 OCR 转写 |
| 图版页 | 每个图号的字段、图片、原图注摘要、校订状态 | 跨书综合理论页 |
| 术语表 | 中文名、英文名、拉丁名、别名、差异说明 | 简短词典残片 |
| 图例库 | 按主题、云类、现象索引图号 | 只有链接入口的列表 |
| 参考资料页 | 来源角色、可信度、校订规则 | 简单书单 |
