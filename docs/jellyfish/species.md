# 代表物种索引

本页先根据《水母花园》整理常见或有代表性的水母和相似类群。后续会逐条扩展为独立物种页。

| 中文名 | 线索 | 备注 |
| --- | --- | --- |
| 海月水母 | 透明圆伞、短触手、常见 | 已建条目：[海月水母](moon-jellyfish.md)。 |
| 多管水母 / 水晶水母 | 透明、可生物发光 | 与水母发光蛋白、绿色荧光蛋白相关。 |
| 夜光游水母 | 粉红或红色，触手可很长 | 成群出现时需要注意蜇伤风险。 |
| 罗盘水母 | 伞面有放射状线条 | 外观看起来像罗盘。 |
| 霞水母 | 大型水母，触手多 | 部分种类体型很大，毒性和具体种有关。 |
| 狮鬃水母 | 极大型冷水水母 | 常被列为最大水母代表之一。 |
| 桶水母 | 白色或带蓝色边缘，体型较大 | 《水母花园》描述其形态像肺，松软有弹性。 |
| 仙后水母 | 倒立生活，与虫黄藻共生 | 伞状体朝下、触手朝上，常见于热带浅海。 |
| 地中海水母 / 蛋黄水母 | 伞面像煎蛋 | 毒性较弱，但具体接触风险仍需校订。 |
| 炮弹水母 | 外形像炮弹，游动较快 | 在部分地区作为食用水母。 |
| 雪茄水母 | 小型，使用下沉触手捕食 | 有“埋伏式”捕食策略。 |
| 澳大利亚箱水母 | 高危险箱水母 | 医疗和安全信息必须引用权威资料后再扩展。 |
| 火水母 | 小型箱水母线索 | 《水母花园》提醒其蜇伤疼痛。 |
| 灯塔水母 | 可逆转生命周期 | 常被称为“永生水母”，需避免过度神化。 |
| 淡海栉水母 | 栉水母，不是刺胞动物水母 | 重要入侵生态案例。 |
