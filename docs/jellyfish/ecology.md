# 生态影响与公众观察

水母并不只是“好看”或“会蜇人”的动物。它们参与海洋食物网，也能反映海洋环境变化。某些海域水母数量突然增加，会影响渔业、旅游、海水取水设施和生态安全。

## 水母暴发

水母暴发指某一水域中水母在短时间内大量出现。暴发可能由多种因素共同触发：

- 水螅体阶段长期潜伏，在合适条件下大量释放幼体。
- 过度捕捞减少鱼类竞争者和捕食者。
- 鱼卵、鱼苗和小型甲壳动物被水母大量摄食，进一步影响鱼类恢复。
- 港口、码头、防波堤等人为硬基质为水螅体提供附着位置。
- 船舶压舱水和船体附着可能帮助水母或相似浮游生物扩散。
- 海水升温、盐度变化和污染可能改变水母适宜栖息范围。

## 生态角色

| 角色 | 说明 |
| --- | --- |
| 捕食者 | 捕食浮游动物、鱼卵、鱼苗等。 |
| 被捕食者 | 海龟、翻车鱼等动物会摄食水母。 |
| 指示生物 | 某些水母分布可提示水团、海流或环境条件。 |
| 物质循环参与者 | 大量水母死亡或排泄物可成为细菌和其他生物利用的有机物来源。 |

## 公众观察

公众科学项目常鼓励海边观察者拍摄并上报水母记录。一次有效记录至少应包含：

- 日期和时间。
- 地点，尽量精确到海滩、港口或经纬度。
- 照片，最好包含整体、伞面、触手和尺度参考。
- 数量：单只、零散、多只、成片。
- 水况：潮汐、风浪、透明度、是否有死亡个体。
- 是否有人被蜇伤，但不要公开个人隐私信息。

## 记录原则

不要徒手捞取或接触不明水母。对高风险海域，应遵守当地海滩警示、救生员建议和医疗机构指导。
