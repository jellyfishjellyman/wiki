# 水母的基础概念

水母不是单一物种，而是一组以胶质身体、浮游生活和刺胞捕食为显著特征的海洋动物通称。日常所说的“水母”常常混用多个类群，本 Wiki 先把范围限定在刺胞动物中的典型水母，并单独记录容易混淆的栉水母等相似动物。

## 核心特征

| 特征 | 说明 |
| --- | --- |
| 胶质身体 | 身体含水量高，外观透明或半透明，质地柔软。 |
| 伞状体 | 多数水母具有圆伞形或钟形的上半部分，用于收缩排水和推进。 |
| 触手和刺胞 | 触手、垂管或伞状体表面常有刺胞，可麻痹猎物，也可能蜇伤人。 |
| 辐射对称 | 俯视常像雨伞或辐条轮，身体结构围绕中央轴线展开。 |
| 浮游生活 | 多数水母水平移动能力有限，常借助水流、风浪和海流扩散。 |

## 为什么水母值得记录

水母已经在地球上存在了很长时间，常被视为古老动物类群的代表。它们分布范围广，从热带到极地、从近岸到深海都可能出现。水母还与海洋生态变化关系密切：水温、盐度、食物链结构、港口设施、船舶压舱水和过度捕捞，都可能影响水母数量和分布。
