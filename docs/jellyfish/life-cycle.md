# 水母的生活史

许多水母的生活史并不是简单的“出生、长大、死亡”，而是在浮游的水母体和附着的水螅体之间转换。理解这一点，有助于解释为什么某些年份几乎看不到水母，某些时候又会突然大量出现。

## 典型阶段

| 阶段 | 生活方式 | 说明 |
| --- | --- | --- |
| 水母体 | 浮游 | 常见的伞状水母，多为有性世代。 |
| 受精卵 / 浮浪幼体 | 浮游后沉降 | 由有性繁殖产生，可寻找附着位置。 |
| 水螅体 | 固着 | 像微小水螅，附着在海底、岩面、港口设施或其他硬基质上。 |
| 横裂体 | 固着分裂 | 在合适条件下横向分裂，像一叠盘子。 |
| 碟状幼体 | 幼小水母 | 从横裂体释放，继续发育为成体水母。 |

## 为什么会突然暴发

当温度、食物、盐度或其他环境条件合适时，附着的水螅体可以在短时间内大量释放幼小水母。人们看到的水母“突然出现”，往往不是凭空发生，而是前期水螅体长期潜伏和无性繁殖的结果。

## 例外与复杂性

并非所有水母都严格遵循同一套生活史。有的类群可直接发育，有的能通过出芽等方式产生小水母。灯塔水母还以能在压力条件下逆转发育阶段而闻名，因此常被称为“永生水母”，但这个说法在科普中容易被夸张，具体机制仍应按物种和研究条件区分。

## 记录建议

- 如果观察到大量水母，应记录时间、水温、潮汐、天气、风向和水域类型。
- 如果研究展缸或附着基质，需留意水螅体和幼体阶段。
- 对“消失”和“突然出现”的判断，应结合多年记录，不宜只凭单次目击。
