# 海月水母

海月水母是常见水母之一，因透明、近圆形的伞状体而得名，也是水族馆展示和公众观察中较容易遇到的类型。

## 基本特征

- 伞状体透明，整体外观像一轮淡色月亮。
- 触手短而纤细，伞缘常显得细碎或参差。
- 可蜇人，但本页暂不把毒性强弱做绝对判断，后续需要结合具体种、地区和医学资料校订。
- 《水母花园》将海月水母列为常见水母，并提到它有较强的自我修复能力。

## 观察要点

- 先看伞状体是否透明、圆盘状。
- 再看伞缘触手是否短而密。
- 记录水体环境、数量、是否成群、是否有幼体或水螅体线索。

## 还需补充

- 学名和中文名对应。
- 全球分布和中国近海记录。
- 与相近海月水母复合群种类的区分。
- 水族馆饲养条件和展缸观察记录。
