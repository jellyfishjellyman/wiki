# 水母蜇伤处理

本页只整理一般性科普原则，不替代医生诊断、急救电话、救生员或当地公共卫生部门建议。不同水母毒性差异很大，处理方式也可能因地区和物种而不同。

## 一般原则

《水母花园》给出的基础处理思路是：

1. 先离开海水，避免继续接触水母或触手。
2. 如果皮肤上残留触手，应尽量清除。
3. 冲洗时优先使用海水，不要用淡水刺激尚未发射的刺胞。
4. 如疼痛、肿胀、瘙痒明显，或出现全身反应，应尽快寻求医疗帮助。

## 不建议草率采用的做法

- 不要用淡水直接冲洗不明水母蜇伤处。
- 不要揉搓伤处。
- 不要凭民间偏方处理严重蜇伤。
- 不要忽视呼吸困难、胸痛、意识异常、大面积蜇伤、儿童或基础病患者的风险。

## 高风险情形

以下情况应优先联系当地急救或海滩救援人员：

- 怀疑为箱水母等高毒性水母。
- 剧烈疼痛快速加重。
- 出现呼吸困难、头晕、恶心、心悸、意识模糊等全身症状。
- 蜇伤面积大，或伤者是儿童、老人、孕妇、心血管疾病患者。

## 说明

不同地区对醋、热水浸泡、止痛和抗毒血清等处理建议可能不同。本页只保留通用风险提示，具体处理应以当地救援、医疗机构和海滩安全指引为准。
