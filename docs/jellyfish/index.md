# 水母

水母是海中的云：它们以透明的身体漂游、收缩、散开，把海水中的流动、光线和生命循环显现出来。

这里整理水母相关资料，包括形态结构、生活史、分类边界、代表物种、生态影响、蜇伤处理和展馆观察笔记。

## 资料说明

本区先以《水母花园》建立基础框架，并逐步参考《水母之书》等资料扩展。涉及分类、毒性和医学处理的内容会尽量保持谨慎，避免把尚未核对的信息写成确定结论。
