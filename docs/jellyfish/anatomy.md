# 水母的形态结构

水母的外形常被比作漂浮的伞。这个比喻虽然简单，但对初步识别很有用：上半部分是伞状体，伞缘和下方连接触手、垂管、口腕等结构。

## 主要结构

| 结构 | 位置 | 功能或识别意义 |
| --- | --- | --- |
| 伞状体 | 身体上半部分 | 通过收缩排水形成推进力；形状可呈圆伞、钟形、盘形或箱形。 |
| 中胶层 | 伞状体内部胶质层 | 使身体有弹性，帮助伞状体在收缩后恢复形状。 |
| 垂管 | 伞状体下方中央 | 与口部相关，参与摄食。 |
| 触手 | 伞缘或身体下方 | 捕食、防御，常带刺胞；长度差异很大。 |
| 刺胞 | 触手、垂管或体表 | 发射含毒结构，用于麻痹猎物；也是蜇伤来源。 |
| 感觉结构 | 伞缘或特化结构 | 部分水母可感知光、重力或方向；箱水母更复杂。 |

## 对称方式

水母多呈辐射对称。它们没有像鱼、昆虫或哺乳动物那样明显的前后方向，这与它们的生活方式有关：水母主要漂浮、垂直升降和等待猎物接触触手，并不依赖快速追逐。

## 游泳方式

多数水母通过伞状体规律收缩，把伞内的水推出去，身体向相反方向移动。它们可以上下移动，但水平方向往往更多受水流影响。少数类群，如部分箱水母和根口水母，水平游动能力更强。

## 识别记录建议

- 拍摄伞状体正面、侧面和伞下结构。
- 记录触手长度、数量感、颜色和是否成束。
- 记录游动方式：垂直升降、水平游动、倒置停留，还是随水流漂移。
- 避免徒手接触，即使个体已经死亡，残留刺胞仍可能造成刺激。
