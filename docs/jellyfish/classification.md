# 分类与相似类群

日常语境里的“水母”常包含多个长得相似但亲缘不同的类群。本 Wiki 暂时采用实用分类：先区分典型刺胞动物水母，再单列栉水母、被囊动物等相似浮游生物。

## 实用分类框架

| 类群 | 是否属于典型水母整理范围 | 说明 |
| --- | --- | --- |
| 钵水母纲 | 是 | 许多常见大型水母属于此类，具有明显水母体阶段。 |
| 箱水母 | 是，单独标注风险 | 伞状体呈箱形，感觉结构和游动能力较复杂，部分种类毒性很强。 |
| 水螅纲水母 | 是，需注明 | 包括一些小型水母和管水母相关类群；灯塔水母属于水螅纲。 |
| 栉水母动物门 | 相似类群，不并入刺胞动物 | 外形像水母，但不具刺胞，使用栉板运动，分类上不同。 |
| 被囊动物亚门浮游类群 | 相似类群 | 某些透明漂浮动物可能被误认作水母。 |

## 水母与栉水母

栉水母看起来常像透明胶质水母，但它们属于栉水母动物门，不是刺胞动物。淡海栉水母就是一个重要例子：它不是典型刺胞动物水母，却因入侵黑海、摄食鱼卵鱼苗和小型甲壳动物而成为生态影响案例。

## 写条目时要避免的问题

- 不把所有透明漂浮动物都归为水母。
- 不只根据“像伞”或“像果冻”判断分类。
- 物种页应尽量同时写中文名、学名、所属类群和是否有刺胞。
- 对箱水母、僧帽水母等高风险对象，应把风险提示与分类信息分开写清楚。
